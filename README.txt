Guía de Uso de la Aplicación de Carrito de Compras
¡Bienvenido a nuestra aplicación de carrito de compras! Esta aplicación te permite gestionar
 tu lista de compras de manera eficiente y realizar pagos mediante diferentes métodos. 
 A continuación, te detallamos cómo utilizar las distintas funcionalidades:

                                1. Explorar y Seleccionar Productos:

Explorar la Página Principal:

Navega por la página principal para visualizar los productos disponibles.

Agregar Productos al Carrito:

Haz clic en los productos que desees comprar. Se añadirán al carrito y verás el contador
de artículos en la barra de tareas.


                                2. Visualizar y Editar el Carrito de Compras:

Ir al Carrito:

Utiliza la opción en la barra de tareas para acceder al carrito y ver los artículos que has seleccionado.
Aumentar o Reducir Cantidad de Productos:

En el carrito, usa los botones "+" o "-" junto a cada producto para ajustar la cantidad.
Eliminar Productos del Carrito:

Utiliza el botón "Eliminar" para quitar productos no deseados de tu lista.


                                 3. Selección del Método de Pago:

Seleccionar el Método de Pago:

Elige entre "Pago con Tarjeta" o "Transferencia" en el menú desplegable del método de pago.


                                4. Pago con Tarjeta:

Datos para Pago con Tarjeta:
Si seleccionas "Pago con Tarjeta", completa los campos requeridos con la información de 
tu tarjeta: número, fecha de vencimiento y código de seguridad (CVV).


                                5. Pago por Transferencia:

Datos para Transferencia:

Si prefieres "Transferencia", introduce el CBU del destinatario y el nombre del destinatario en los campos correspondientes.


                                6. Realizar la Compra:

Confirmar la Compra:
Una vez que hayas ingresado los datos necesarios, haz clic en el botón "Realizar Compra" para finalizar
el proceso de compra.


Notas Importantes:
Validación de Datos:
Verifica que todos los campos requeridos estén completos antes de realizar la compra para evitar inconvenientes.
Atención al Cliente:
Si tienes alguna duda o encuentras algún problema, no dudes en contactarnos a través de nuestro 
servicio de atención al cliente.

¡Listo! Ahora estás preparado para utilizar nuestra aplicación de carrito de compras. 
                        ¡Disfruta de tu experiencia de compra!
